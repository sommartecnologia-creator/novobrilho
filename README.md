# JP NOVO BRILHO - Website Corporativo

Website corporativo profissional desenvolvido para a **JP NOVO BRILHO, LDA.**, empresa angolana especializada em serviços técnicos e gráficos.

## 🌟 Sobre o Projeto

Este website foi desenvolvido em **HTML5**, **CSS3** e **JavaScript** puro, oferecendo uma experiência moderna, responsiva e otimizada para empresas que buscam presença digital profissional em Angola.

### Identidade Visual

O design utiliza as cores da bandeira angolana como inspiração:
- **Vermelho (#CE1126)** - Cor primária, representando energia e determinação
- **Preto (#000000)** - Cor secundária, representando força e elegância
- **Dourado (#FFCD00)** - Cor de destaque, representando excelência e qualidade

## 🚀 Como Usar

### Visualização Local

1. **Clone ou descarregue o projeto**
2. **Abra o arquivo `index.html`** em qualquer navegador moderno
3. **Navegue pelas páginas** usando o menu de navegação

### Publicação Online

#### Opção 1: Via tab "Publish" (Recomendado)
- Acesse a aba **Publish** nesta plataforma
- Clique em **Publish** para publicar o site
- Receberá uma URL pública automaticamente

#### Opção 2: Upload manual para hospedagem
1. Faça upload de todos os arquivos para seu servidor web
2. Certifique-se de manter a estrutura de pastas intacta
3. Configure o domínio conforme necessário
